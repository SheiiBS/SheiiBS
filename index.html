--- layout: page title: Gestión de Configuración permalink: / translate_en: /en/ --- El cómo enfrentar los cambios es todo un arte.

La Gestión de Configuración es una disciplina dentro de la Ingeniería de Software que permite coordinar el desarrollo de software para minimizar la confusión. Es el arte de identificar, organizar y controlar las modificaciones que sufre el software.

La meta es siempre maximizar la productividad minimizando errores.
